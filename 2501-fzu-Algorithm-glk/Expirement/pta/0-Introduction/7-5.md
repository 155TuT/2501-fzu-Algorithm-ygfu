# 7-5 整数A+B

分数 10

作者 喵喵-lxq

单位 江西师范大学

本题目要求读入2个整数A和B，然后输出它们的和。

## 输入格式

请在这里写输入格式。例如：输入在一行中给出2个绝对值不超过1000的整数A和B。

## 输出格式

请在这里描述输出格式。例如：对每一组输入，在一行中输出A+B的值。

## 输入样例

在这里给出一组输入。例如：

```cpp
18 -299
```

## 输出样例

在这里给出相应的输出。例如：

```cpp
-281
```

- 代码长度限制 16 KB
- 时间限制 400 ms
- 内存限制 64 MB
- 栈限制 8192 KB

