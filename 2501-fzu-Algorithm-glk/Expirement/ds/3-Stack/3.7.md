# 3.7 火车

## 实验任务

TonyY 等火车无聊的时候，会去观察火车的排列，有一天他思考这么一个问题，火车总站的火车只能进站，要出站的话只能先出最后进站的那辆车，那么知道火车的进站顺序，能不能把它的出站顺序调整成火车站想要的呢？

## 数据输入

输入第一行为一个正整数 n 表示火车辆数(编号 1-n)（1<=n<=9）。

然后为两个长度为 n 的序列，表示火车的进站顺序和出站顺序。每辆火车刚好进出站各一次。

## 数据输出

如果可以调整，输出 Yes 和出入站顺序。

如果不能，输出 No。

## 输入示例

```cpp
3
123
321
```

## 输出示例

```cpp
Yes
in
in
in
out
out
out
```

## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int main() {
    ios::sync_with_stdio(0);
    cin.tie(0);

    int n;
    if (!(cin >> n)) return 0;
    string inSeq, outSeq;
    cin >> inSeq >> outSeq;

    stack<char> st;
    vector<string> ops;

    int i = 0;
    for (int j = 0; j < n; j++) {
        while ((st.empty() || st.top() != outSeq[j]) && i < n) {
            st.push(inSeq[i]);
            ops.push_back("in");
            i++;
        }

        if (!st.empty() && st.top() == outSeq[j]) {
            st.pop();
            ops.push_back("out");
        } else {
            cout << "No\n";
            return 0;
        }
    }

    cout << "Yes\n";
    for (auto &op : ops) cout << op << '\n';

    return 0;
}
```
