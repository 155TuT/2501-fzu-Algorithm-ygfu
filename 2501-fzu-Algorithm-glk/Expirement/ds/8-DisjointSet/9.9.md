# 9.9 道路

## 实验任务

某省调查城镇交通状况，得到现有城镇道路统计表，现给出每条道路连接的城镇编号，问当前的道路设计方案是否合理。

合理的方案为任意两个城镇之间可以相互到达，有且只有一条通路。

## 数据输入

输入a b(1<=a,b<=10,000)，表示城镇 a 和城镇 b 连通。输入包含多组数据，每组数据以 0 0 结束。

整个文件以 -1 -1 结尾

## 数据输出

如果方案合法，输出"Yes",否则输出"No"。

## 输入示例

```cpp
6 8 5 3 5 2 6 4 5 6 0 0
8 1 7 3 6 2 8 9 7 5 7 4 7 8 7 6 0 0
3 8 6 8 6 4 5 3 5 6 5 2 0 0
1 2 3 4 0 0
-1 -1
```

## 输出示例

```cpp
Yes
Yes
No
No
```

## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int parent_[10005];
bool used[10005];

int findp(int x) {
    if (parent_[x] == x) return x;
    return parent_[x] = findp(parent_[x]);
}

int main() {
    ios::sync_with_stdio(0);
    cin.tie(0);

    int a, b;
    while (1) {
        for (int i = 1; i <= 10000; i++) {
            parent_[i] = i;
            used[i] = false;
        }
        bool cycle = false;
        int components = 0;

        if (!(cin >> a >> b)) return 0;
        if (a == -1 && b == -1) break;

        auto ensure = [&](int x) {
            if (!used[x]) {
                used[x] = true;
                parent_[x] = x;
                components++;
            }
        };

        if (a == 0 && b == 0) {
            cout << "Yes" << '\n';
            continue;
        }

        ensure(a);
        ensure(b);
        int ra = findp(a), rb = findp(b);
        if (ra == rb) {
            cycle = true;
        } else {
            parent_[ra] = rb;
            components--;
        }

        while (cin >> a >> b) {
            if (a == 0 && b == 0) break;

            ensure(a);
            ensure(b);
            ra = findp(a);
            rb = findp(b);
            if (ra == rb) {
                cycle = true;
            } else {
                parent_[ra] = rb;
                components--;
            }
        }

        if (cycle) {
            cout << "No" << '\n';
        } else {
            if (components == 1 || components == 0) cout << "Yes" << '\n';
            else cout << "No" << '\n';
        }
    }

    return 0;
}
```
