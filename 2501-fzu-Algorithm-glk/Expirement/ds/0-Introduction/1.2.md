# 1.2 哥德巴赫猜想问题

## 实验任务

验证哥德巴赫猜想。计算给定的大偶数可以表示为多少对素数之和。例如，大偶数 10 可以表示为 2 对素数 3，7 和 5，5 之和。

## 数据输入

输入数据只有一行，一个大偶数 N（2<=N<=30000）。

## 数据输出

输出计算出的相应的素数分解数。

## 输入示例

```cpp
10
```

## 输出示例

```cpp
2
```

## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int main() {
    ios::sync_with_stdio(0);
    cin.tie(0);

    int N; 
    if (!(cin >> N)) return 0;
    vector<int> prime(N + 1, 1);
    if (N >= 0) prime[0] = 0;
    if (N >= 1) prime[1] = 0;
    for (int i = 2; i * i <= N; i++) if (prime[i]) {
        for (int j = i * i; j <= N; j += i) prime[j] = 0;
    }
    int cnt = 0;
    for (int p = 2; p <= N / 2; p++) {
        if (prime[p] && prime[N - p]) cnt++;
    }
    cout << cnt << '\n';
    return 0;
}
```
