# 10.32 最小生成树

## 问题描述

给定平面上的n个点，添加一些边使得任何两个点之间都能相互到达，求添加的边的长度总和最小为多少

## 数据输入

第一行一个正整数n(1<=n<=5000)，表示序列有n个点，
接下来有n行，每行两个整数xi和yi，（-1000000 ≤ xi,yi ≤ 1000000），表示n个点的坐标

## 数据输出

添加的边的长度总和的最小值，四舍五入保留两位小数

## 输入示例

```cpp
4
0 0
1 2
-1 2
0 4
```

## 输出示例

```cpp
6.47
```

## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int main() {
    ios::sync_with_stdio(0);
    cin.tie(0);

    int n;
    if (!(cin >> n)) return 0;
    vector<double> x(n), y(n);
    for (int i = 0; i < n; i++) {
        cin >> x[i] >> y[i];
    }

    const double INF = 1e100;
    vector<double> dist(n, INF);
    vector<bool> used(n, false);

    dist[0] = 0.0;
    double ans = 0.0;

    for (int i = 0; i < n; i++) {
        int u = -1;
        double best = INF;
        for (int j = 0; j < n; j++) {
            if (!used[j] && dist[j] < best) {
                best = dist[j];
                u = j;
            }
        }
        used[u] = true;
        ans += dist[u];

        for (int v = 0; v < n; v++) {
            if (!used[v]) {
                double dx = x[u] - x[v];
                double dy = y[u] - y[v];
                double d = sqrt(dx * dx + dy * dy);
                if (d < dist[v]) dist[v] = d;
            }
        }
    }

    cout.setf(ios::fixed);
    cout << setprecision(2) << ans << '\n';

    return 0;
}
```
